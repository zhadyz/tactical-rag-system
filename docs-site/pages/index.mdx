## Quick Start

Get Apollo running in under 5 minutes:

```bash
# Clone the repository
git clone https://github.com/yourusername/apollo-rag.git
cd apollo-rag

# Launch with Docker Compose (includes GPU support)
docker compose up -d

# Upload your first document
curl -X POST http://localhost:8000/upload \
  -F "file=@document.pdf" \
  -H "Authorization: Bearer your-api-key"

# Ask a question
curl -X POST http://localhost:8000/query \
  -H "Content-Type: application/json" \
  -d '{"query": "What is the main topic?"}'
```

See the [Getting Started](/getting-started) guide for detailed instructions.
